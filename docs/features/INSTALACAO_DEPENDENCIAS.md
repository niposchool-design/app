# 📦 Instalação de Dependências para Gamificação

## Dependências Necessárias

Para que o sistema de gamificação funcione completamente, instale:

```bash
npm install @radix-ui/react-progress @radix-ui/react-slot class-variance-authority
```

Ou com yarn:

```bash
yarn add @radix-ui/react-progress @radix-ui/react-slot class-variance-authority
```

## Pacotes Instalados

- **@radix-ui/react-progress**: Componente de barra de progresso acessível
- **@radix-ui/react-slot**: Utilitário para composição de componentes
- **class-variance-authority**: Utilitário para variantes de classes CSS

## Verificação

As seguintes dependências JÁ estão instaladas (necessárias também):

- ✅ clsx
- ✅ tailwind-merge
- ✅ lucide-react

## Instalação Completa

Execute este comando único:

```bash
npm install @radix-ui/react-progress @radix-ui/react-slot class-variance-authority
```

Depois:

```bash
npm run dev
```

E acesse:
- http://localhost:4000/alunos/gamificacao
