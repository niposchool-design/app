## 📈 Distribuição de Aulas por Nível

### Iniciante (Aulas 0-6)
- **7 aulas** (23% do curso)
- Foco: Base rítmica, iniciação instrumental, alfabetização básica
- Metodologias: Orff, Suzuki, Kodály, Dalcroze

### Intermediário (Aulas 7-12, 21, 23)
- **8 aulas** (27% do curso)
- Foco: Leitura, repertório, técnica, criatividade
- Metodologias: Kodály, Gordon, Experiências Brasileiras, Musical Futures

### Avançado (Aulas 13-20, 22, 24)
- **10 aulas** (33% do curso)
- Foco: Harmonia, composição, tecnologia, improvisação, performance
- Metodologias: Berklee, PRESTO, Gordon, Musical Futures, Waldorf

### Consolidação/Todos os Níveis (Aulas 25-29)
- **5 aulas** (17% do curso)
- Foco: Ensaios, avaliação, apresentação final

## 📊 Banco de Dados - Campos Necessários

### Tabela: `aulas`
```sql
- id (UUID)
- numero (INT)
- titulo (TEXT)
- data_programada (DATE)
- objetivo_didatico (TEXT)
- nivel (ENUM: iniciante, intermediario, avancado, todos)
- modulo (TEXT) -- ex: "BLOCO 1: Fundação"
- metodologia_principal (TEXT) -- ex: "Orff Schulwerk"
- metodologias_secundarias (TEXT[]) -- array
- status (ENUM: a_fazer, em_preparacao, concluida, cancelada)
- resumo_atividades (TEXT)
- desafio_alpha (TEXT)
- formato (ENUM: presencial, online, hibrido)
- duracao_minutos (INT)
- professor_responsavel_id (FK)
```

### Tabela: `aula_materiais`
```sql
- id (UUID)
- aula_id (FK)
- tipo (ENUM: pdf, video, partitura, audio, formulario, slide, link)
- titulo (TEXT)
- descricao (TEXT)
- url (TEXT)
- arquivo_path (TEXT)
- ordem (INT)
- obrigatorio (BOOLEAN)
- visivel_antes_aula (BOOLEAN)
```

### Tabela: `aula_pre_requisitos`
```sql
- id (UUID)
- aula_id (FK) -- aula atual
- pre_requisito_aula_id (FK) -- aula necessária antes
- obrigatorio (BOOLEAN)
```

### Tabela: `aluno_progresso_aula`
```sql
- id (UUID)
- aluno_id (FK)
- aula_id (FK)
- status (ENUM: nao_iniciada, em_andamento, concluida)
- data_inicio (TIMESTAMP)
- data_conclusao (TIMESTAMP)
- porcentagem_completa (INT)
- desafio_enviado (BOOLEAN)
- desafio_aprovado (BOOLEAN)
- nota_auto_avaliacao (INT 1-5)
```
